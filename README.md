#  School Marks Management System

This is a simple and powerful **School Management System** built using HTML, CSS, and JavaScript.  
It helps teachers manage student marks, calculate totals, percentages, and generate rankings automatically.

---

##  Features

- Add student names and marks
- Automatic total calculation
- Percentage calculation
- Class ranking system (1st, 2nd, 3rd…)
- Simple and easy-to-use interface
- Works online or offline in browser

---

##  Purpose

The purpose of this system is to help schools and teachers:
- Reduce manual calculation errors
- Save time when marking exams
- Improve student performance tracking

---

##  How to Use

1. Open the website or `index.html` file
2. Enter student name
3. Enter subject marks
4. Click "Add Student"
5. View results and ranking automatically
